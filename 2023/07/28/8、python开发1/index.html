文件操作

字符串类型：在程序中用于表示文字信息，本质上是unicode编码中的二进制

字节类型：可表示文字信息，本质上是utf-8/gbk等编码的二进制。也可表示原始二进制(图片，文件等信息)

读文件

1.打开文件(rb,表示读取文件原始的二进制)

file_object = open(‘info.txt’,mode=’rb’)

2.读取文件

data = file_object.read()

3.关闭文件

file_object.close()

print(data)

4.直接读字符串内容

text = data.decode(“utf-8”)

print(text)

或者直接file_object = open(‘info.txt’,mode=’rt’,encoding=’utf-8’)

写文件

文件打开模式

  • 只读：r、rt、rb

    存在，读

    不存在，报错

  • 只写：w、wt、wb

    存在，清空再写

    不存在，创建再写

  • 只写：x、xt、xb

    存在，报错

    不存在，创建再写

  • 追加写：a、at、ab

    存在，尾部追加

    不存在，创建再写

  • 读写：

    r+、rb+，起始位置

    w+、wb+，清空然后起始位置

    x+、xb+，新文件起始位置

    a+、ab+，末尾位置

data.seek(0)让光标回到开头
