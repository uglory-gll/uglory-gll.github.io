ACTF2020 新生赛(Include)

文件不在显示部分，可能在注释部分，用到文件包含漏洞，知识点php://filter可以获取指定文件源码。当其与文件包含函数结合时，php://filter流会被当作php文件执行。所以我们一般对其进行编码，阻止其不执行，从而读取任意文件源代码格式：xxx.php?xxx=php://filter/convert.base64-encode/resource=xxx.php

ACTF2020 新生赛(Exec)

很明显可能存在命令注入漏洞，用;结束前面的语句，然后后面进行自己的代码

强网杯 2019(随便注)

过滤了关键字，我们可以用堆叠查询，同时用show tables,show columns from table_name，这里学到一个新知识点，表名为数字时，要用反引号包起来查询

payload:0’;rename table words to word1;rename table 1919810931114514 to words;alter table words add id varchar(255);alter table words change flag id varchar(255);#

这是因为words的id列会显示出来，我们构造语句把flag想办法变成words表里面的id列

SUCTF 2019(EasySQL)

set sql_mode=pipes_as_concat是将||的作用由or变为拼接字符串

这里要猜测后台语句，我是猜不到，select $_POST[‘query’] || flag from Flag，我们传参*,1即可

GXYCTF2019(Ping Ping Ping)

用$IFS$1代替空格查看index.php它过滤了很多东西 包括一些字符还有整个flag

在linux系统中，反引号是作为内联执行，输出查询结果的内容。比如用ls查询出index.php。那么ls就代表了index.php这个文件。那么我们就可以使用cat命令查看index.php的内容，而这一道题就可以使用cat$IFS$1ls 这个命令来输出flag了payload：**?ip=1;cat$IFS$1`ls`**
