hexo博客搭建

发表于2022-09-16|更新于2022-09-17|小知识
|字数总计:420|阅读时长:1分钟|阅读量:

  1. 安装node.js，用node -v,npm -v进行检测是否安装成功

  2. 安装git，用git –version进行检测是否安装成功

  3. 安装hexo，npm install hexo-cli -g，hexo -v检测是否安装成功

  4. 去github建立一个仓库

    • 这里可以改名

    • 新建一个仓库

    • 这两个名字一定要相等

  5. 看看ssh是否安装过，默认是安装过得

    接下来输入ssh-keygen -t rsa -C “邮件地址”，然后一直回车

  6. C:\Users\Administrator\.ssh这里有ssh的文件,打开这个文件

    复制里面内容，去github，点击setting，头像那里，再找到ssh

    然后看看是否安装成功

    ssh -T git@github.com，yes

  7. 在要写博客的文件夹

    hexo init#初始化，报错可能是github国外网不稳定的原因

    然后这里已经多了很多内容

    hexo s#生成可以访问的本地页面

    这里就可以访问了

  8. 修改配置文件

    deploy:

    type: git

    repository: https://github.com/你的名字/仓库的名字.git

    branch: 分支

    将这个内容代替最后的内容

  9. 安装hexo-deployer-git自动部署发布工具(一样的文件夹打开git bush):

    npm install hexo-deployer-git –save

  10. hexo g

    hexo d

    报错，这是因为是国外网的原因

    git config –global http.proxy http://127.0.0.1:1080

  11. 看看仓库，已经成功

  12. ping xxx.github.io找到自己blog的ip

  13. 去买个域名

  14. 设置CNAME

    source文件夹下面创建CNAME文件(没有后缀名)

    写你买的域名

    然后hexo d

  15. 买的域名设置两条解析

    • A对应ip
    • CNAME对应xxx.github.io

  16. 进入github，点击仓库，setting，page

  17. 用域名去访问
